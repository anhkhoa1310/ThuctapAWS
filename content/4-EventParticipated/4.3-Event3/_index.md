# Key Presentation Highlights

## 1. Opening Session
**Speaker: Hieu Nghi**

The workshop began with an introduction to the objectives of the event and the importance of preparing for a career in the technology industry.

The speaker emphasized that success in IT does not depend solely on technical knowledge but also on continuous learning, communication skills, teamwork, and a proactive mindset.

Some important messages included:

- Build a clear learning roadmap.
- Focus on solving problems instead of memorizing syntax.
- Learn through real projects.
- Develop both technical and soft skills simultaneously.

The opening session motivated students to become lifelong learners and actively prepare for future career opportunities.

---

## 2. Personal Project Sharing – URL Shortener
**Speakers: Đinh Trung Kiên & Nguyễn Minh Thọ**

The speakers introduced their personal software project, a **URL Shortener**, demonstrating how a seemingly simple application actually combines many important software engineering concepts.

The project included:

- URL encoding and decoding
- Database management
- RESTful API development
- User authentication
- Click statistics
- Deployment process

The speakers also explained how personal projects help students improve programming skills while building an attractive portfolio for future recruitment.

One valuable lesson shared was:

> "A strong portfolio with practical projects often creates a greater impression than academic transcripts alone."

---

## 3. Career Sharing – Analytics Engineer
**Speaker: Nguyễn Đạt**

This session introduced the Analytics Engineer role, which connects Data Engineering with Business Intelligence.

The speaker shared daily responsibilities such as:

- Collecting raw data.
- Cleaning and transforming datasets.
- Building data warehouses.
- Creating dashboards.
- Supporting business decisions using data analysis.

Several technologies commonly used in this field were introduced:

- SQL
- Python
- Power BI
- Tableau
- dbt
- Apache Airflow
- Cloud Data Platforms

The presentation highlighted that SQL proficiency and logical thinking are essential foundations for pursuing a career in data analytics.

---

## 4. Workplace Culture
**Speaker: Nguyễn Mạnh Cường**

This presentation focused on professional working culture within technology companies.

The speaker explained that technical ability alone is not enough for long-term success. Professional attitude and communication skills are equally important.

Topics discussed included:

- Responsibility at work.
- Time management.
- Professional communication.
- Respect for colleagues.
- Continuous self-improvement.
- Receiving and giving constructive feedback.

This session reminded students that employers highly value professionalism and teamwork.

---

## 5. Career Sharing – DevOps Engineer
**Speaker: Trương Hoàng Trọng**

The DevOps session was the most comprehensive technical presentation of the workshop.

The speaker explained how DevOps bridges software development and IT operations by improving automation, collaboration, and deployment efficiency.

### DevOps Mindset

The presentation introduced several important principles:

- Automation First
- Continuous Integration
- Continuous Delivery
- Infrastructure as Code
- Monitoring & Observability
- Continuous Improvement

### CI/CD Pipeline

A standard software delivery pipeline was demonstrated:

- Source Code Management (Git)
- Build Automation
- Automated Testing
- Security Scanning
- Docker Image Build
- Deployment
- Monitoring Production Systems

### DevOps Technologies

The speaker also introduced many technologies commonly used in industry:

- Git
- GitHub
- GitHub Actions
- Jenkins
- Docker
- Kubernetes
- Terraform
- Linux
- AWS
- Prometheus
- Grafana

Besides technical knowledge, the speaker shared valuable real-world experiences about deployment failures, monitoring incidents, automation strategies, and collaboration between development and operations teams.

---

## 6. Q&A Session

The workshop concluded with an interactive discussion session between students and speakers.

Participants had opportunities to ask questions regarding:

- Career orientation.
- Learning roadmaps.
- Internship preparation.
- Technical interviews.
- Cloud Computing.
- DevOps career opportunities.

Listening to the speakers' practical advice helped students better understand industry expectations and prepare more effectively for their future careers.

# Key Takeaways

## Technical Knowledge

The workshop expanded my understanding of several important IT career paths, particularly Analytics Engineering and DevOps Engineering.

Among all sessions, the DevOps presentation impressed me the most because it demonstrated how automation, cloud computing, and collaboration can significantly improve software development efficiency.

---

## Career Orientation

This event helped me better understand the differences between software development, data engineering, and DevOps.

It also encouraged me to continue learning:

- Docker
- Kubernetes
- GitHub Actions
- Linux
- AWS
- CI/CD
- Infrastructure as Code

These technologies are becoming increasingly important in modern software development.

---

## Soft Skills & Professional Mindset

Besides technical knowledge, I learned that communication, teamwork, responsibility, and continuous learning are essential qualities for every software engineer.

Professional attitude is often just as important as technical ability.
