| Information | Details |

### Event Objectives

- Review AWS Cloud knowledge through an interactive team competition.
- Learn about AWS Security Agent services and their practical applications in enterprise cloud environments.
- Understand the importance of Service Level Agreements (SLA) and monitoring for maintaining highly available cloud systems.

### Key Presentation Highlights

#### 1. AWS Knowledge Competition

The workshop began with an exciting AWS knowledge competition between **Team KLKAT from HUTECH** and **Team Ngũ Đại Hiệp from SGU** using **AWS Question Set 10**. The questions covered a wide range of AWS topics, including cloud architecture, networking, storage, compute services, security, and best practices.

Our team, **KLKAT**, successfully achieved **Top 1**. Besides reinforcing the AWS knowledge learned throughout the First Cloud AI Journey program, the competition also strengthened teamwork, communication skills, and the ability to solve technical problems under time pressure.

---

#### 2. AWS Security Agent Services

**Speaker:** Nguyễn Tuấn Thịnh

The second session introduced AWS Security Agent services and demonstrated how organizations can continuously monitor cloud resources to detect potential security threats.

Key topics included:

- The role of Security Agents in cloud environments.
- Collecting logs and security events automatically.
- Continuous monitoring for cloud infrastructure.
- Detecting abnormal behaviors and suspicious activities.
- Live demonstrations showing how security monitoring supports administrators in protecting AWS resources.

The presentation highlighted the importance of proactive monitoring and automation in modern cloud security operations.

---

#### 3. Service Level Agreement (SLA) & Monitoring

**Speaker:** Nguyễn Huỳnh Sơn

This session focused on maintaining service quality and system availability through effective monitoring.

Important topics included:

- Understanding AWS Service Level Agreements (SLA).
- Measuring service availability and uptime.
- Monitoring infrastructure performance.
- Configuring alerts for abnormal system behavior.
- Live demonstrations of monitoring dashboards and operational metrics.

The speaker emphasized that monitoring is an essential part of cloud infrastructure because it allows engineers to detect issues early and minimize service downtime.

---

#### 4. AWS Cloud Certification Roadmap

**Speaker:** Ngô Lê Tấn Huy

The final presentation shared practical experiences about AWS certification preparation.

The speaker introduced:

- AWS certification pathways.
- Recommended learning roadmaps.
- Effective study methods.
- Examination tips and tricks.
- Common mistakes made by beginners.

Students were encouraged to practice regularly using AWS Free Tier while combining theoretical learning with hands-on projects.
