### Week 1 Objectives:

* Become familiar with the First Cloud AI Journey (FCAJ) internship program and its learning roadmap.
* Understand the internship regulations, communication channels, and reporting process.
* Meet mentors and teammates and understand the internship expectations.
* Prepare the working environment for upcoming AWS training and project development.
* Install and configure the required development tools.

### Tasks to be carried out this week:

| Day | Task | Start Date | Completion Date | Reference Material |
| --- | -------------------------------------------------------------------------------------------------------------------------------- | ---------- | --------------- | ------------------ |
| Monday | Attend the internship orientation session, meet mentors and teammates, and receive an overview of the FCAJ program, internship roadmap, and expected outcomes. | 20/04/2026 | 20/04/2026 | FCAJ Orientation |
| Tuesday | Learn the internship regulations, communication channels, reporting procedures, attendance policy, and internship evaluation criteria. | 21/04/2026 | 21/04/2026 | Internship Handbook |

### Week 1 Achievements:

* Successfully completed the internship orientation program and became familiar with the overall learning roadmap of the First Cloud AI Journey program.

* Understood the internship regulations, including:
  * Working schedule.
  * Attendance and leave request procedures.
  * Weekly reporting requirements.
  * Communication etiquette.
  * Professional conduct during the internship.

* Learned the communication workflow between mentors, teammates, and interns using the official communication platforms.

* Understood the internship milestones, including:
  * AWS learning modules.
  * Hands-on laboratory sessions.
  * Technical blog writing.
  * Workshop participation.
  * Project implementation.
  * Final presentation and evaluation.

* Successfully installed and configured the required development tools:
  * Git.
  * GitHub Desktop.
  * Visual Studio Code.
  * Hugo Extended.
  * Docker Desktop.
  * AWS CLI.

* Verified that the development environment was functioning properly and ready for AWS learning modules and project development.

* Received an introduction to the internship project:

  **Online Pharmacy Website & AI Medical Consultation Chatbot System based on AWS Serverless**

* Understood the overall system objectives and the technologies that would be used throughout the internship, including AWS Serverless services, cloud computing, and AI applications.

* Built confidence in using the internship resources and prepared for the following technical learning weeks.

### Knowledge Gained:

* Internship regulations and professional working etiquette.
* Internship workflow and reporting procedures.
* Team communication and collaboration process.
* Git-based source code management.
