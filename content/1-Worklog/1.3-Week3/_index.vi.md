### Mục tiêu tuần:

* Tiếp tục học các dịch vụ cốt lõi của AWS thông qua AWS Learning Modules.
* Củng cố kiến thức hạ tầng Cloud bằng các bài Hands-on Labs.
* Tìm hiểu kiến thức về mạng trên AWS và Infrastructure as Code (IaC).
* Chuẩn bị nền tảng Cloud phục vụ cho dự án thực tập.

### Kết quả đạt được:

* Hoàn thành các AWS Learning Modules liên quan đến hạ tầng mạng trên AWS.
* Hoàn thành Hands-on Labs bao gồm:
  * Amazon VPC
  * Public & Private Subnet
  * Route Table
  * Internet Gateway
  * Security Group
* Hiểu cách triển khai hạ tầng AWS bằng AWS CloudFormation.
* Thực hành Infrastructure as Code với AWS CDK.
* Sử dụng AWS CLI để quản lý tài nguyên AWS.
* Nâng cao kiến thức về kiến trúc mạng trên AWS.
* Chuẩn bị nền tảng hạ tầng Cloud cho giai đoạn phát triển dự án.

### Kiến thức học được:

* Kiến trúc Amazon VPC.
* AWS CLI.
* Infrastructure as Code (CloudFormation và AWS CDK).
* Thành phần mạng trên AWS.
* Cách triển khai tài nguyên Cloud.
* Các thực hành tốt trong việc xây dựng hạ tầng trên AWS.
